# Health2You Legal Site Export

Static files for a minimal GitHub Pages legal/support site.

## Files

- `index.html` — landing page linking to Privacy Policy and Support.
- `privacy.html` — Health2You Privacy Policy.
- `support.html` — Health2You Support page.

## Suggested GitHub Pages Repository

Suggested repo name:

`health2you-legal`

## How to Publish on GitHub Pages

1. Create a public GitHub repository named `health2you-legal`.
2. Copy the contents of this folder into the repository root:
   - `index.html`
   - `privacy.html`
   - `support.html`
3. Commit and push to the repository.
4. In GitHub, open **Settings → Pages**.
5. Set **Source** to deploy from the `main` branch root.
6. Wait for GitHub Pages to publish.

## Expected URLs

Replace `<github-username>` with the GitHub account or organization name:

- `https://<github-username>.github.io/health2you-legal/privacy.html`
- `https://<github-username>.github.io/health2you-legal/support.html`

Use those URLs in App Store Connect for the Privacy Policy URL and Support URL.
